# Banking_Program.py
Banking Program (Python CLI)

A simple command-line banking system built with Python that allows users to create secure accounts, manage their money, and track transactions. The program simulates basic banking operations such as deposits, withdrawals, and balance checks while maintaining a transaction history for each account.

This project demonstrates core programming concepts including object-oriented programming (OOP), input validation, error handling, and basic financial transaction management.

Features

🔐 PIN-Protected Accounts
Each user account is secured with a 4-digit PIN.

👤 Multiple Account Support
Users can create and manage multiple accounts.

💰 Deposit Money
Add funds to an account with input validation.

💳 Withdraw Money
Withdraw funds with balance checks to prevent overdrawing.

📊 Balance Display
View the current balance at any time.

📜 Transaction History
All deposits and withdrawals are recorded with:

Transaction type

Amount

Date and time

Balance after transaction

🔑 Login System with Security

Users get 3 PIN attempts before being returned to the main menu.
